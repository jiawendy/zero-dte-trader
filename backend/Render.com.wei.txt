Deployment Guide (Render.com)
This guide explains how to deploy the 0DTE Trader App to Render.com for free.

Prerequisites
A GitHub account.
A Render account.
Git installed on your computer.
Step 1: Push Code to GitHub
Create a new repository on GitHub (e.g., zero-dte-trader).
Open a terminal in the project folder:
cd /Users/marklu/GoogleAntigravity/zero-dte-trader
git init
git add .
git commit -m "Initial commit"
git branch -M main
# If you get "error: remote origin already exists", run this instead:
# git remote set-url origin https://github.com/YOUR_USERNAME/zero-dte-trader.git
git remote add origin https://github.com/YOUR_USERNAME/zero-dte-trader.git
git push -u origin main
Step 2: Deploy Backend (Web Service)
Log in to Render Dashboard.
Click New + -> Web Service.
Connect your GitHub repository.
Configure the service:
Name: zero-dte-backend
Root Directory: backend
Runtime: Python 3
Build Command: pip install -r requirements.txt
Start Command: uvicorn main:app --host 0.0.0.0 --port $PORT
Environment Variables (Advanced -> Add Environment Variable):
TRADIER_API_KEY: xKMr2rl3WAAP9Gx4uYLPXT9IE1PV
GOOGLE_API_KEY: (Paste your Google API Key here)
PYTHON_VERSION: 3.11.0 (optional, but recommended)
Click Create Web Service.
Copy the Backend URL (e.g., https://zero-dte-backend.onrender.com) once it's live.
Step 3: Deploy Frontend (Static Site)
Click New + -> Static Site.
Connect the same GitHub repository.
Configure the site:
Name: zero-dte-frontend
Root Directory: frontend
Build Command: npm install && npm run build
Publish Directory: dist
Environment Variables:
VITE_API_URL: (Paste your Backend URL from Step 2, e.g., https://zero-dte-backend.onrender.com)
Click Create Static Site.
Step 4: Share!
Once the frontend is deployed, Render will give you a URL (e.g., https://zero-dte-frontend.onrender.com). Share this link with others!
